Política de Privacidade

Esta política descreve como o Support Bolt coleta e utiliza os dados necessários para operar seus recursos de automação, integração e notificação no Discord.

1. Dados coletados

Podemos armazenar identificadores técnicos, IDs de servidor, configurações administrativas, tokens autorizados por OAuth e metadados de perfis conectados quando isso for necessário para a funcionalidade solicitada.

2. Finalidade do uso

Os dados são utilizados para autenticar contas, consultar APIs autorizadas, gerar anúncios automáticos, manter configurações por servidor e garantir a continuidade operacional do sistema.

3. Compartilhamento

O Support Bolt não comercializa dados pessoais. Processamento externo só acontece quando necessário para executar a funcionalidade autorizada pelo administrador.

4. Armazenamento e segurança

Credenciais e tokens são mantidos em ambiente de servidor e usados exclusivamente pelo backend autorizado, com medidas razoáveis de segurança.

5. Retenção

Os dados permanecem armazenados enquanto forem necessários para operar o serviço ou até remoção pelo administrador responsável.
